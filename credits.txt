Código e Ferramentas
A arquitetura do projeto foi desenvolvida com um conjunto de ferramentas modernas e eficientes, focadas em produtividade e boas práticas:

React & TypeScript: O projeto foi construído sobre a poderosa base do React para criar interfaces de usuário dinâmicas e reativas, com a segurança de tipos do TypeScript garantindo um código mais robusto e escalável.
Tailwind CSS: Para a estilização, foi utilizado o Tailwind CSS, um framework utility-first que permite a construção de designs complexos e responsivos diretamente no HTML/JSX, agilizando o desenvolvimento e mantendo a consistência visual.
Vite: Todo o ambiente de desenvolvimento e o processo de build são gerenciados pelo Vite, proporcionando uma experiência de desenvolvimento extremamente rápida com Hot Module Replacement (HMR) instantâneo
