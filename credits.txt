##Recursos Visuais
Fontes
A tipografia deste projeto utiliza as seguintes fontes, ambas licenciadas sob a SIL Open Font License, Version 1.1:
Inter: Criada por Rasmus Andersson e contribuidores. Disponível em rsms/inter.
Fira Code: Criada por Nikita Prokopov e contribuidores. Disponível em tonsky/FiraCode.

Ícones
O ícone de menu (hambúrguer) e o de código são parte da biblioteca Material Symbols, oferecida pelo Google.
Os ícones de Machine Learning foram criados por VectorPortal e estão disponíveis na plataforma Flaticon.

Código e Ferramentas
React & TypeScript
O projeto foi construído sobre a poderosa base do React e a segurança de tipos do TypeScript, utilizando Vite como ferramenta de build.

CSS Reset
Utilizamos o CSS Reset v2.0 por Eric Meyer, um recurso de domínio público essencial para garantir a consistência visual do projeto em diferentes navegadores.
